Installation¶

install using pip:

pip install git+ssh://git@github.com/ajmirsky/mirskutils.git

or if you’re interested in staying up-to-date with the latest, install the source:

pip install -e git+ssh://git@github.com/ajmirsky/mirskutils.git#egg=mirskutils
